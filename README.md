# pikvm-onecloud

PiKVM for onecloud

# How to use?

Copy all files in this repo to `armbian-build/userpataches/`.

Like this:
```
armbian-build
├── cache
├── config
├── extensions
├── lib
├── output
├── packages
├── patch
├── tools
└── userpatches
    ├── customize-image.sh
    ├── atf
    ├── kernel
    │   ├── archive
    │   ├── meson-current
    │   ├── meson-edge
    │   └── meson-legacy
    ├── misc
    └── u-boot
```
